# Hybrid Controls

Bayesian dynamic borrowing methods for clinical trials using commensurate priors. Supports multiple endpoint types including binary, time-to-event, and continuous outcomes.

## Overview

This project implements advanced Bayesian methods for incorporating historical or concurrent control information into clinical trials. Dynamic borrowing allows optimal use of auxiliary data while controlling for potential biases.

## Features

- **Commensurate Priors** - Hierarchical Bayesian approach for dynamic borrowing
- **Multiple Endpoint Types** - Binary, time-to-event, and continuous endpoints
- **Flexible Design** - Adaptable to various trial designs
- **Bayesian Inference** - Full Bayesian posterior computation
- **Real-World Application** - Designed for practical clinical trial scenarios

## Technologies

- R
- JAGS (Just Another Gibbs Sampler)
- Bayesian Statistical Methods

## Methods Implemented

- Dynamic borrowing with commensurate priors
- Posterior inference and MCMC sampling
- Multiple endpoint handling
- Prior sensitivity analysis

## Getting Started

1. Install R and JAGS
2. Clone the repository
3. Load the R scripts and JAGS models
4. Adapt examples to your trial design
